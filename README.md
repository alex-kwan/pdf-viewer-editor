# pdf-viewer-editor
A tool that can be hosted on the web so people can view and manipulate PDFs

## Milestone 1 Frontend (Implemented)

The milestone 1 viewer is available in `web/`.

### Run locally

1. `cd web`
2. `npm install`
3. `npm run dev`
4. Open the local Vite URL shown in terminal (usually `http://localhost:5173`)

### Features in this milestone

- Upload PDF by file picker or drag and drop
- Render PDF pages with PDF.js
- Previous/next and jump-to-page navigation
- Zoom in/out plus fit-width and fit-page
- Clickable thumbnail sidebar
- Basic invalid file and password-protected file error handling
